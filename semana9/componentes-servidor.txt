Sistema operativo:
Es el software principal del computador o servidor. Administra los recursos del sistema y permite que las aplicaciones funcionen correctamente.

Node.js:
Es un entorno de ejecución para JavaScript que permite crear y ejecutar aplicaciones del lado del servidor de forma rápida y eficiente.

Aplicación server.js:
Es el archivo donde se encuentra el código del servidor. Define las rutas, las respuestas y la lógica de la aplicación.

Puerto:
Es un punto de comunicación por el que el servidor recibe solicitudes de los usuarios. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros que almacenan información sobre la ejecución del servidor, errores y eventos importantes para facilitar el monitoreo y la solución de problemas.
